Customer Churn Analysis Dashboard built using Power BI to analyze customer retention, churn patterns, contract types, payment methods, internet services, and customer demographics.

# Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn in a telecommunications company using Power BI. The objective is to identify factors influencing customer churn and provide insights that help improve customer retention.

## Tools Used

* Power BI
* Power Query
* DAX
* Data Visualization

## Key Metrics

* Total Customers: 7,043
* Churned Customers: 1,869
* Churn Rate: 26.54%

## Dashboard Features

### Customer Demographics

* Gender Distribution
* Partner Status Analysis
* Dependents Distribution

### Churn Analysis

* Contract Type vs Churn
* Payment Method vs Churn
* Internet Service vs Churn

### Customer Retention

* Customer Tenure Distribution

## Key Insights

* Month-to-Month contract customers have the highest churn.
* Electronic Check users show higher churn rates.
* Fiber Optic customers churn more than DSL customers.
* Long-term contracts improve customer retention.

## Project Outcome

The dashboard helps businesses identify high-risk customer groups and make data-driven decisions to improve customer retention strategies.
